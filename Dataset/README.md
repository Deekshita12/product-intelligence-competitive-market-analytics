# 📂 Dataset

This folder contains the **first part of the original public e-commerce product dataset** used for the **Product Intelligence & Competitive Market Analytics** project.

The complete dataset has been divided into two folders (**Dataset** and **Dataset 2**) to improve repository organization and simplify file management on GitHub.

---

# 📊 Dataset Overview

The files in this folder contain raw product-level information collected from a public e-commerce marketplace. These datasets served as the primary input for the ETL pipeline, PostgreSQL data warehouse, SQL analysis and Power BI dashboards.

Each dataset contains product-related information such as:

- Product Name
- Main Category
- Sub Category
- Selling Price
- Original Price
- Discount Percentage
- Customer Ratings
- Number of Ratings
- Product Image URL
- Product URL
