# 📂 Dataset 2

This folder contains the **second part of the original public e-commerce product dataset** used for the **Product Intelligence & Competitive Market Analytics** project.

Together with the **Dataset** folder, it represents the complete raw dataset used throughout the ETL pipeline, PostgreSQL data warehouse development, SQL analytics and Power BI dashboard creation.

---

# 📊 Dataset Overview

The files in this folder contain raw product-level information collected from a public e-commerce marketplace.

Each dataset includes information such as:

- Product Name
- Main Category
- Sub Category
- Selling Price
- Original Price
- Discount Percentage
- Customer Ratings
- Number of Ratings
- Product Image URL
- Product URL
